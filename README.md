# helloword
Contenedores de node js, mongo db y rabbitmq

## Requirements

* Docker <br>
* Node js <br>
* Mongo db <br>
* Rabbitmq <br>

## Start the project

### Install

Clone the repo

```bash
git clone https://github.com/03Jenry/helloword.git
```

Move to the folder

```bash
cd helloword
```

run the install.sh file

```bash
./install.sh
```
